# LP-UTN
Se realizó un sitio web integrando todos los contenidos vistos durante la cursada, adhiriendo conocimientos propios y aprendidos durante la misma. Se utilizó el editor de código Visual Studio Code junto a algunos códigos suministrados desde la página suministrada por la cursada, Bootstrap. 
El sitio en cuestión refiere a una página web de una banda de música. En ella se podrá encontrar: 
1) Un indice donde nos recibe la navbar con diferentes enlaces a otras pages del website, un vídeo colocado mediante un iframe sobre la banda en cuestión. Datos ficticios sobre próximos shows al igual que artículos a la venta, culminando con un footer con la información correspondiente del copyright y los enlaces a las redes sociales de la banda.
2) Una page de "Nosotros" donde se hace una mínima reseña sobre la banda encuestión y se adjunta las redes sociales de los tres integrantes más antiguos.
3) Enlaces hacía "Servicios" donde  se manifestaban diferentes funcionalidades. Las mismas redireccionan a una página de mantenimiento para que tenga un correcto cierre y sea funcional.
4) Por último, una sección de "Contactanos", donde se suministra un formulario de contacto para distintas eventualidades.
